Windows上配置Total Commander + Listary

Total Commander是一款功能强大的文件管理器，提供了双窗格布局和丰富的插件支持。而Listary则是一款搜索工具，它可以极大提升文件查找和操作的效率。当两者结合使用时，能有效提高文件管理的工作效率。

1. 安装Total Commander

1.1 下载Total Commander

首先，从Total Commander官网下载最新版本的安装程序：

  1. 访问Total Commander下载页面。

  2. 选择Windows的安装程序，并下载。

1.2 安装Total Commander

下载完成后，双击安装程序并按照以下步骤安装：

  1. 选择安装语言，点击“OK”。

  2. 接受许可协议，点击“Next”。

  3. 选择安装目录，建议使用默认路径。点击“Next”继续。

  4. 配置快捷方式和文件关联设置，点击“Next”。

  5. 最后点击“Install”完成安装。

安装完成后，你可以启动Total Commander，开始熟悉它的基本操作。

2. 安装Listary

2.1 下载Listary

从Listary官方网站下载Listary最新版本：

  1. 访问Listary下载页面。

  2. 点击“Download”按钮下载适用于Windows的版本。

2.2 安装Listary

与Total Commander安装类似，下载完成后，双击安装程序并按照提示操作：

  1. 接受许可协议并点击“Next”。

  2. 选择安装目录，点击“Next”。

  3. 配置安装选项，默认设置即可，点击“Next”。

  4. 点击“Install”进行安装，最后点击“Finish”完成。

安装完成后，Listary会自动运行，并常驻系统托盘。

3. 配置Total Commander与Listary结合使用

Total Commander和Listary各有其优势，两者结合可以提升搜索和文件管理效率。下面介绍如何将两者完美结合。

3.1 打开Total Commander的目录选择

要让Listary完美结合Total Commander，需要开启Total Commander的目录选择功能：

  1. 打开Total Commander，进入配置窗口（快捷键 Alt + P 或点击顶部菜单中的“配置”）。

  2. 选择“操作”选项卡，确保“支持通过命令行启动路径”选项已勾选。这个选项允许Total Commander接受来自外部应用的路径跳转命令。

3.2 配置Listary与Total Commander集成

要让Listary搜索结果可以直接跳转到Total Commander：

  1. 右键点击系统托盘中的Listary图标，选择“选项”进入配置页面。

  2. 在左侧导航栏中选择“应用程序”，然后点击右侧的“添加”按钮。

  3. 在弹出的窗口中，浏览到Total Commander的安装目录，并选择TOTALCMD.EXE，点击“打开”。

  4. 这时Listary会将Total Commander添加为支持的应用程序。接着，你可以配置具体的操作行为：

    • 在“命令”选项中，输入 %1，这表示将搜索结果的文件或文件夹路径传递给Total Commander。
    • 你还可以选择是否在新标签页中打开该路径，或覆盖当前标签页。

3.3 测试集成效果

完成配置后，可以通过以下步骤测试：

  1. 按下 Win + S 或双击 Ctrl 键，调出Listary搜索框。

  2. 输入文件或文件夹的名称，Listary会快速展示搜索结果。

  3. 选中一个结果并按 Enter，此时Listary会直接在Total Commander中打开对应的文件或文件夹。

4. 提高Total Commander使用效率的其他插件

Total Commander还支持丰富的插件，以下是几个推荐的插件，可以进一步提升使用体验：

  1. Everything 插件：为Total Commander提供快速文件搜索支持，类似Listary，但搜索速度极快。可以从 插件页面 下载并安装。

  2. TC Media Player 插件：用于在Total Commander中直接播放音频或视频文件，方便快速预览媒体文件。

  3. FileInfo 插件：为文件提供更多详细信息显示，适用于需要查看文件的元数据、权限等高级信息的场景。

5. 常见问题解决

5.1 Listary无法跳转到Total Commander

如果配置好后发现Listary无法打开Total Commander：

  1. 检查Total Commander的路径是否正确添加。

  2. 确保Total Commander已启用“支持通过命令行启动路径”功能。

5.2 搜索速度较慢

如果你感觉Listary的搜索速度较慢，可以尝试：

  1. 在Listary配置页面的“搜索”选项中，设置索引文件夹，避免Listary搜索不必要的路径。

  2. 安装Everything插件以提高搜索速度。

6. 总结

通过结合使用Total Commander和Listary，你可以显著提升Windows下的文件管理和搜索效率。Total Commander提供了强大的文件管理功能，而Listary的快速搜索与Total Commander的双窗格管理相结合，让你的日常文件操作更加高效。如果你是多任务处理的用户，强烈推荐这两款工具的组合。
